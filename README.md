This is my personal website repository.
Check it out here: https://friedrich.fschr.me

An important aspect was minimalist yet impressive design.
This meant little text and smooth animations that did not get in one's way.

Feel free to get inspired.

## Production

The production build reads blog posts from:

```env
VITE_API_BASE_URL=https://api.friedrich-fischer.de
```

The GitHub FTP workflow builds the Vite app and deploys the generated `dist/` directory.
