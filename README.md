# simple-personal-website

Personal site built with Vite and React.

## Requirements

- Node.js 22+
- npm

## Install

```bash
npm install
```

## Run the dev server

```bash
npm run dev
```

Vite is configured with `--host`, so it will listen on the network as well as localhost.

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```

## Notes

- Static assets live in `public/`.
- The CV download buttons serve the PDF files from `public/`.
